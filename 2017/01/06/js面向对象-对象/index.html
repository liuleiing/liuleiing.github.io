1. 对象

JavaScript的所有类型，都源于一个Object，也就是我们常说的万物皆对象

对象作用：代码的复用，封装各种库

对象的属性：实体的静态特征，通过var定义

对象的方法：实体的动态功能，通过function定义

2.创建自定义对象

2.1 工厂方式

工厂方式创建对象实际上就是对内置系统对象创建方法对一种封装

2.2 构造函数的方式

new关键字的作用：

1.new 去调用一个对象的时候，里面的this指向这个函数创建出来的对象（obj）

2.会默认返回函数创建出来的对象（obj）

2.3 对象字面量的形式

3. JSON

JSON是js对象的一种简写形式，格式很严格，key值必须带””，value可以是任意类型(数字、字符串、逻辑值、数组、对象、null、函数或匿名函数)

JSON的作用：数据交互

4. 对象的引用和比较

对象的引用，引用类型赋值（地址），基本类型赋值（拷贝）

对象的比较,对象不仅要比较值 还要比较引用 也就是说看地址是不是同时指向一个地方的

5. 原型prototype

prototype是构造函数的一个属性，用来存放一些公共方法。原型

js是基于原型的程序
