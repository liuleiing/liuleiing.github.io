css学习笔记 background属性

Share

内容总结

  • background

1. background

background是复合属性，包括了background-(color|image|repeat|attachment|position)

书写与顺序无关，可以多个用空格隔开，也可以写一个。复合属性会覆盖单个属性，所以单个属性一般放在复合属性的下面

1.1 background-color

语法:background-color:transparent|color

transparent:表示背景色透明

1.2 background-image

语法:background:none|url()

1.3 background-repeat

语法： background-repeat :repeat| no-repeat| repeat-x| repeat-y

repeat-x:表示背景图横向平铺

repeat-y : 背景图像在纵向平铺

需要先设置背景图片(background-image)之后设置才是否平铺

1.4 background-position

语法:position : top | center | bottom | left | center | right

如果只指定了一个值，该值将用于横坐标。纵坐标将默认为50%。第二个值是纵坐标。 值可以写英文，也可以写数字，甚至可以是负数。常用于 CSS sprites（CSS精灵）

该属性定位不受对象的补丁属性(padding)设置影响

1.5 background-attachment

语法： background-attachment :scroll| fixed

scroll : 背景图像是随对象内容滚动

fixed : 背景图像固定

2. background-CSS3

css3中新增的背景属性

2.1 background-size

设置背景图片的大小

语法： = [ | | auto ]{1,2} | cover | contain

cover:撑满整个容器,背景图像有可能超出容器

contain:将背景图像等比缩放到宽度或高度与容器的宽度或高度相等，背景图像始终被包含在容器内

2.2 background-origin

设置图片的原点位置，即background-position的起始位置

语法:background-origin:padding-box|border-box|content-box

padding-box:默认值。
border-box:原点位置从border开始
content-box:原点位置从content开始
