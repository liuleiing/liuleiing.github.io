2. Mongodb安装

MongoDB是一个基于分布式文件存储的数据库，由c++语言编写。文件存储格式为BSON(json的扩展)
BSON是一种类似json的二进制形式的存储格式，简称Binary JSON，它和JSON一样，支持内嵌的文档对象和数组对象，但是BSON有JSON没有的一些数据类型，如Date和BinData类型

MongoDB官网地址

MongoDB下载地址

由于我的操作系统是win32,所以以下配置步骤适用于win32的系统

  1. 以管理员身份运行cmd，
  2. 要想执行mongodb的命令，需要cd到他的安装目录下面的bin文件夹下
  3. 在想建立数据库盘上(比如：d盘)创建文件夹mongodb,并且建立子目录db和log
  4. 把mongodb文件夹的只读属性去掉

  5. 执行命令：mongod –dbpath “d:\mongodb\db” –logpath “d:\mongodb\log\MongoDB.log” –logappend (win32系统)

    mongod –storageEngine mmapv1 –dbpath “d:\mongodb\db” –logpath “d:\mongodb\log\MongoDB.log” –logappend (win64系统)

  6. 以管理员身份运行cmd，进入bin文件夹，执行命令: mongod –dbpath “d:\mongodb\db” –logpath “c:\mongodb\log\MongoDB.log” –logappend –install –serviceName “MongoDB”
  7. 启动mongodb服务:net start MongoDB

3. Mongodb基本使用

以下命令都在Mongodb的bin目录下的mongo.exe中执行。（因为没有配置环境变量）

作用 命令
显示数据库列表 >show dbs
创建(使用)数据库 >use dbname
显示集合 >show collections
增加数据 >db.collectionsName.save({“key”:”value”})
>db.collectionName.insert({“key”:”value”})
删除文档 >db.collectionsName.remove({})
删除集合 >db.users.drop()或>db.runCommand({“drop”,”users”})
删除数据库 >db.runCommand({“dropDatabase”: 1})

4.Mongodb条件查询
查找key=value的数据 db.collection.find({“key”:value})
key>value db.collection.find({“key”:{$gt:value}})
key<value db.collection.find({“key”:{$lt:value}})
key>=value db.collection.find({“key”:{$gte:value}})
key<=value db.collection.find({“find”:{$lte:value}})
value1<key<value2 db.collection.find({“key”:{$lt:value2,$gt:value1}})
key<>value db.collection.find({“key”:{$ne:value}})
取模运算 key%10==1 db.collection.find({“key”:{$mod:[10,1]}})
key等于[1,2,3]中任何一个 db.collection.find({“key”:{$in:[1,2,3]}})
key不等于[1,2,3]中任何一个 db.collection.find({“key”:{$nin:[1,2,3]}})
key对应的值的数量是1（值必须是数组） db.collection.find({“key”:{$size:1}})
true返回存在字段key的数据，false返回不存在字段key的数据 db.collection.find({“key”:{$exists:true\false}})
符合两个条件中任意一个的数据 db.collection.find({$or:[{“key1”:value1},{“key2”:value2}])
内嵌对象中的值匹配 db.collection.find({ “key.subkey” :value })

mongdb其他操作
排序，1代表升序，-1代表降序 db.collection.find().sort({“key1”:-1,”key2”:1})
添加索引,1代表升序，-1代表降序 db.collection.ensureIndex({“key”:1})
控制返回结果数量，如果参数是0，则当作没有约束 db.collection.find().limit(5)
控制返回结果跳过多少数量 db.collection.find().skip(5)
返回结果集的条数 db.collection.find().count(true\false)
格式化 db.conllections.find().pretty()
